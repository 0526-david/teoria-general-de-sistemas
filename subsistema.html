subsistemas

Un subsistema es un sistema que es parte de otro sistema mayor que lo contiene.

En otras palabras, un subsistema es un conjunto de elemento interrelacionados que, en sí mismo, es un sistema, pero a la vez es parte de un sistema superior.

CARACTERÍSTICAS DE LOS SUBSISTEMAS

Los subsistemas tienen 9 características: propósito, limite, contorno, componentes, interacciones, recursos, ingresos, egresos y subproductos, si no las tienen no se pueden clasificar como y un subsistema otra determinante en la habilidad de funcionar como un sistema.

EJEMPLOS DE SUBSISTEMAS

  • Sistema principal : un humano

    Subsistemas: sistema respiratorio, sistema circular, sistema nervioso, sistema reproductor, etc.

  • Sistema principal: una computadora

    Subsistemas: el microprocesador, la fuente de energía, el disco duro, etc.

  • Sistema principal: un auto

    Subsistemas: el sistema de frenado, el sistema de airbags, el sistema de computadora abordo, etc.

  • Sistema principal: un sistema operativo de computadora

    Subsistemas:el sistema de manejo de ventajas, el sistema de seguridad, el sistema de actualización, el sistema de administración de memoria, etc.

  • Sistema principal: un árbol

    Subsistemas: el sistema circulatorio del árbol, cada una de sus células vegetales, el sistema de la fotosíntesis en sus hojas, etc.
